# **App Name**: HighClass HR Automation

## Core Features:

- Dynamic QR Code Generation: Generate secure, dynamic QR codes daily for employee check-in/out.
- Mobile Check-in/out: Enable employees to record attendance by scanning QR codes with their phone's back camera.
- Real-time Attendance Tracking: Track employee attendance in real time.
- Automated Deduction Calculations: Automatically calculate delays and deductions.
- One-Click Payroll Generation: Generate accurate monthly payroll with a single click, aided by tool to consider edge cases.
- Firebase Realtime Database Storage: Store all records in Firebase Realtime Database for instant synchronization and secure access.
- Progressive Web App (PWA) Support: Make application installable to mobile devices with push notifications for announcements.

## Style Guidelines:

- Primary color: Deep blue (#293462), conveying professionalism and trust.
- Background color: Light blue-gray (#DDE2EC), a neutral backdrop that enhances readability.
- Accent color: Soft lavender (#B7A2E6) for highlights and interactive elements, creating a gentle contrast.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet approachable aesthetic. Note: currently only Google Fonts are supported.
- Auto-responsive design to adapt seamlessly to different screen sizes.
- Use simple, clear icons to represent actions and data, improving UX for Arabic speakers and global users.
- Subtle animations to indicate loading or confirmation of actions, enhancing user feedback without distracting.
